Real-Time Multiplayer Auction Engine
A web-based multiplayer simulator designed to replicate a live auction environment. Built to handle concurrent users, real-time database updates, and complex budget constraints.

Key Features:

Live Multiplayer Bidding: Synchronized real-time auction state across all connected clients.

Structured Player Database: A comprehensive dataset of players categorized by specific bidding pools and roles.

Dynamic State Tracking: Live visibility into team progress, including remaining purse, squad size constraints, and overseas player limits.

Host Control Mechanisms: A dedicated pre-game lobby for team selection, plus administrative host controls to manually undo/redo actions and resolve edge cases during live bidding.

Endgame Voting System: A democratic voting phase at the conclusion of the draft to determine the winning manager based on team composition and budget efficiency.

Streamlined UI: A simple, fast-responsive interface engineered for high-speed decision making.
